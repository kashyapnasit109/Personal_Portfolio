# Personal Portfolio

A modern, highly interactive, and visually stunning personal portfolio website built with React, Vite, and Tailwind CSS. 
The site features smooth scrolling via Lenis, advanced animations using GSAP and Framer Motion, and a fully responsive design tailored for a premium user experience.

## ✨ Features

- **Modern Tech Stack**: Built with React 19 and Vite for blazing-fast performance.
- **Immersive Animations**: Utilizes GSAP and Framer Motion for scroll-triggered and element-level animations.
- **Smooth Scrolling**: Integrated with Studio Freight's Lenis for buttery smooth scroll experiences.
- **Beautiful Styling**: Designed with Tailwind CSS for rapid, utility-first styling.
- **Contact Form Integration**: Includes direct email capabilities using EmailJS.
- **Responsive Design**: Flawlessly adapts to mobile, tablet, and desktop viewports.

## 🚀 Quick Start

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kashyapnasit109/Personal_Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Personal_Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
To start the development server, run:
```bash
npm run dev
```
The application will be available at `http://localhost:5173/` (or another port if specified by Vite).

### Building for Production
To build the project for production, run:
```bash
npm run build
```
This will generate an optimized build in the `dist` folder, which can be deployed to any static hosting service.
